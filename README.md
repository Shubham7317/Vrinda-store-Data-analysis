📊 Vrinda Store Annual Report 2022 (Excel Dashboard)
📌 Project Overview

This project is an interactive Sales Dashboard created in Microsoft Excel to analyze the annual performance of Vrinda Store. It provides insights into sales, orders, customer demographics, and sales channels using charts, slicers, and pivot tables.

The dashboard helps in understanding trends and making better business decisions through simple visualizations.

🚀 Features
📅 Monthly Sales & Orders Analysis
👩‍🦰 Sales by Gender (Men vs Women)
📦 Order Status Overview
🌍 Top 5 States by Sales
👥 Age Group vs Gender Analysis
🛒 Sales by Channels (Amazon, Myntra, Ajio, etc.)
🎯 Interactive Filters (Month, Channel, Category)


📊 Dashboard Insights
Women contribute the majority of sales (~100%)
Most orders are successfully delivered
Adult age group dominates purchases (~57%)
Myntra is the leading sales channel
Top states include Telangana, West Bengal, and Haryana

Microsoft Excel
Pivot Tables
Pivot Charts
Slicers & Filters
